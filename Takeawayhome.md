# AI Engineer Take-Home Assignment

## Objective

Build a **To-Do List & Calendar Web Application** that allows users to organize their daily tasks through an interactive calendar interface.

The application should support:
- Authentication
- Task management
- Task status tracking

---

# Functional Requirements

## 1. User Authentication

Implement:

- User Registration
- User Login
- User Logout
- Protected routes/pages

Requirements:

- Each user should only be able to access and manage **their own tasks**.

---

## 2. Calendar

The calendar should allow users to:

- View tasks by date
- Select any date on the calendar
- Display all tasks associated with the selected date
- Create new tasks directly for the selected date

---

## 3. Task Management

Users should be able to:

- Create a task
- Edit a task
- Delete a task
- View task details

Each task should contain:

- **Title**
- **Description**
- **Date**
- **Status**

### Available Statuses

- Not Started
- In Progress
- Done

---

## 4. Dashboard

The dashboard should display:

- Calendar view
- Task list for the selected date
- Summary of tasks grouped by status:
  - Not Started
  - In Progress
  - Done

---

## 5. Database

Persist all user and task data in a database.

> Local storage alone is **not sufficient**.
